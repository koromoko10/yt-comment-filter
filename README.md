## この拡張機能について
特定のパターンに合うYouTubeコメントを非表示にするChrome拡張機能です。<br>
YouTube内でよくある
* **「1コメ！」**
* **「今見てる人いる？」**
* **「今見てる人は流石にいないでしょ」**
* **「今見てる人挙手」**
* **「今いいねついたら流石に怖い」**
* **「〇〇年も見てる人いる？」**
* **「なう(日付)」**
* **「いいね数を〇〇から〇〇にしてしまった罪悪感」**
* **「伝説はここから始まった」**
* **「それってあなたの～」**
* **「jkの〇〇動画上げてます」**
* **「〇〇円もらえる動画上げてます」**
* **「絶対に翻訳したら～」**
* **「草とかつけとけばいいと思ってるでしょ～」**
* **「で？」**
* **「呪いにかかります～」**

などの動画の内容に直接関係のないコメントを非表示にします。<br>

> [!NOTE]
> 返信の非表示はうまく動作しないことがあります。<br> 
> **ぜひIssuesやプルリクお待ちしております！**

## 導入方法
こちらの[リリースページ](https://github.com/koromoko10/yt-comment-filter/releases)からZIPファイルかcrxファイルをダウンロードできます。
### ZIPファイルからインストール
1. あらかじめZIPファイルを展開しておく
2. 拡張機能のページに移動する
3. デベロッパーモードをオンにする
4. `パッケージ化されていない拡張機能を読み込む`から展開したフォルダを選択
5. 必要に応じて拡張機能をピン止めする

### crxファイルからインストール
1. 拡張機能のページに移動する
3. デベロッパーモードをオンにする
4. ダウンロードしたcrxファイルをページにドラッグアンドドロップする
5. 必要に応じて拡張機能をピン止めする

## 非表示リストを編集する
### カスタムユーザーフィルターを設定する
デフォルトで内臓しているリストとは別に**カスタムユーザーフィルター**を設定できます。<br>
ブラウザ右上の拡張機能のアイコンから`設定`をクリックすると、オプション画面に遷移します。<br>
その画面上でカスタムユーザーフィルターの編集が可能です。
### デフォルトフィルターを編集する

> [!WARNING]
> ソースを直接変更するので上級者向けです

デフォルトのフィルタリングが気に入らない時は**直接リストを編集する** ことで変更することができます。<br>
ダウンロードしたZIPファイル内に `content.js` があるのでそれを編集します。<br>
`content.js` 内の　`const commentFilters` の中身が非表示にしている単語リストです。 <br>

> 正規表現で入力します。例：`/単語/i`
